# S3 Sync (многопоточная синхронизация S3 -> S3)

Простое CLI-приложение на Go для синхронизации содержимого между несколькими парами путей в S3. 
Конфигурация задается массивом задач в `config.yaml`. Каждая задача исполняется как отдельный поток (параллельно).

Возможности:
- Копирование объектов из источника в назначение (с сохранением относительного пути).
- Сравнение только по имени (ключу); при совпадении можно либо пропускать, либо перезаписывать (опция overwriteOnNameMatch).
- Опциональное удаление объектов в назначении, которых нет в источнике.
- Параллельный запуск нескольких задач и текстовый прогресс (шкала) по каждой задаче.

## Установка и запуск

Требования:
- Go 1.26+
- Доступ к AWS S3 (переменные окружения, профиль или иные способы аутентификации, поддерживаемые AWS SDK).
